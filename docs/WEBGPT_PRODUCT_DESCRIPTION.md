# WEBGPTとは

**WEBGPT**は、企業向けのAIチャットボットサービスです。Webサイトの情報を自動学習し、24時間365日の即時応答を実現します。夜間や休日のサポート対応を自動化し、**サポート工数の約80%削減**と**応答時間の約80%短縮**を実現できます。

## 主な特徴

- **24時間365日の即時応答**: 夜間や休日も自動で対応し、翌朝のチケット整理が激減します

- **ブランドトーンに合わせた会話**: マニュアルやFAQから学習し、トーン&マナーを崩さずに回答します

- **95言語の自動認識**: ユーザーの言語を自動判定し、自然な表現で返答します

## こんな方におすすめ

- **サポートチームが限られている企業**: 夜間や休日の対応に課題を抱えている

- **24時間対応が必要な企業**: グローバル展開やサービス業で常時サポートが必要

- **開発リソースを割けない企業**: 開発チームに負担をかけずに導入できる

## 主要機能

### 自動学習システム

ベースURLを登録するだけで、クローラーが対象ページを収集し学習に反映します。サイトマップURLや個別URLリストにも対応し、最短数時間でテスト可能です。ページ数が多い場合でも原則1営業日以内にサンプルを提供します。

### 埋め込みウィジェット

発行されたスクリプトを1行貼るだけで、どのCMSでもすぐにチャットが表示されます。WordPressやWixなど主要CMSでも同様に設置でき、開発リソースを必要としません。

### 多言語対応

ユーザーの言語を自動判定し、95以上の言語で自然な表現で返答します。グローバル展開している企業でも、追加設定なしで多言語対応が可能です。

### 会話チューニング

ナレッジや禁止ワードを設定し、応答の質をコントロールできます。会話ログとフィードバックをもとに自動で調整し、必要に応じて人の手でも微調整可能です。

### 継続学習

会話ログをもとに自動でアップデートします。サイトの内容が変わったときは再学習リクエストで対応し、スタジオ以降のプランでは自動再学習機能を搭載予定です。

### 堅牢なセキュリティ

データは暗号化して保存し、アクセス権限も細かく管理します。社内ナレッジを安全に活用でき、プライバシーとセキュリティを確保します。
